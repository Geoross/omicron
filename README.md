## ⚡ WHAT IS OMICRON?
Omicron is a lightweight, powerful desktop utility built to do one thing perfectly: download high-quality video and audio from YouTube. No ads, no web-browser bloat, just a clean interface and fast results. 
## 🛠️ HOW TO USE
1. Run the Omicron executable file.
2. Set your default save directories for Video and Audio using the "Change" buttons in the Kitchen (Settings) area.
3. Paste your desired YouTube URL into the input field (Right-click -> Paste, or hit Ctrl+V).
4. Select your preferred quality from the dropdown (e.g., Crisp 1080p, Potato 480p, Just the Tunes).
5. *(Optional)* If downloading a playlist, check **Playlist Mode 📜**. A menu will pop up allowing you to select exactly which videos you want to keep.
6. Hit **Munch It! 🦖** (or press the Enter key) to add the item to the Oven (queue) and start downloading.

*(Note: Omicron requires an active internet connection to fetch the media streams.)*
